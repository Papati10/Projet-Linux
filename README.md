# Projet Linux

## Description

Mini projet de systèmes d’exploitation réalisé sous Linux.

## Contenu du projet

### 1. Bash

Script d’administration système :

- création de dossiers

- affichage des informations système

- gestion automatique

Fichier :

- admin_systeme.sh

### 2. CPU Scheduling

Comparaison entre :

- SRTF

- Round Robin

Fichiers :

- srtf.py

- round_robin.py

### 3. Docker

Conteneurisation avec Docker Compose.

Fichiers :

- docker-compose.yml

- Dockerfile

- index.php

- init.sql

## Technologies utilisées

- Linux

- Bash

- Python

- Docker

- MySQL

- PHP
